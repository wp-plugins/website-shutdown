== Description ==



A plugin which on activation displays "Website is down for maintenance, Please try again later" to all non-admin users. The admin can still go around any part of the site with no problems and can log-into the admin area even if the plugin is activated.

== Installation ==

1. Upload `shutdown.php` to the `/wp-content/plugins/` directory
2. When you want to shutdown the site activate the plugin
